## Démonstration

> Création d'un repo local

```bash
mkdir 00-git-local &&
cd 00-git-local
```

> Edition d'un (ce) fichier readme.md, en guise de page d'acceuil de mon futur repo

```bash
touch readme.md
nano readme.md
```
